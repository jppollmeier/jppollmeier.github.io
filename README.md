# Markdown Cheat Sheet

## Headers
# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

## Emphasis
*Italic* or _Italic_
**Bold** or __Bold__
**_Bold and Italic_** or *__Bold and Italic__*
~~Strikethrough~~

## Lists
- Bullet list
  - Nested bullet
    - Sub-nested bullet

1. Numbered list
2. Number two
   1. Sub-number

## Links
[Title](https://www.example.com)

## Images
![Alt text](https://www.example.com/image.jpg "Optional title")

## Code
`Inline code` with backticks


## Blockquotes
> Blockquote
>> Nested blockquote

## Tables

| Syntax | Description |
| ----------- | ----------- |
| Header | Title |
| Paragraph | Text |

| Syntax | Description | Test Text |
